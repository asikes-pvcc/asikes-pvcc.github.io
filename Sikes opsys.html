Operating System Definitions

Term Definition
Operating sytem The software that controls the overall operation of a computer
Job queue A storage organization in which jobs are ordered in a first in, first out fashion
FIFO First in, first out
Interactive processing The ability of an executed program to carry on a dialogue with a user through remote terminals
Time-sharing Ability to provide service to multiple users at the same time
Multiprogramming A technique for job execution in which time is divided into intervals and each job is restricted to a single interval
Application software Software which consists of the programs for performing tasks particular to a machine's utilization
System software Software which performs tasks common to computer systems in general and provides the infrastructure the application software requires
Utility software Software units that extend or customize the capabilities of an operating system
Graphical user interface (GUI) An interface in which files, programs, and other objects to be manipulated are represented pictorially on a display
Shell An older style user interface that communicates with users through textual messages
Kernel Internal part of an operating system
Device driver Software units that communicate directly or via controllers to carry out operations of peripheral devices attached to the machine
File manager Coordinates the use of a machine's storage facilities by maintaining records and locations of files stored in mass storage as well as space available for new files
Memory manager Coordinates use of a machine's main memory
Virtual memory Memory that appears to exist due to program rotation (paging) between main memory and mass storage of a device
Boot strapping A procedure prompted by powering on a device which transfers the operating systemm from mass storage to main memory
ROM A type of memory (Read Only Memory) that can only be accessed, not altered
Deadlock the condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling Holding data for output at a later but more convenient time
Super user (or administrator) A user who can alter settings within an operating system, modify critical software packages, adjust privileges, and perform a variety of other maintenance activities not available to other users
Multi-factor authentication Security measure which requires a user to provide several different pieces of indentification to gain access to a device or account
