Chapter 6: Programming Languages

Program Language Term Definitions
Machine Language Sequences of instructions encoded as numeric digits
Debugging The process of locating errors in a program and correcting them
Assembler A program developed to convert mnemonic expressions into machine language instructions
Assembly Language A mnemonic system for representing programs
Translator A Program which translate high level primative programs into machine language programs
Compiler A translator that has to compile machine instructions into short sequences
Interpreter Program which executes instructions as they are translated
Cross-platform Software A program that is both operating system independent and machine independent
Natural Language A language such as English or German that does not require precise, unchanging language and phrasing for comprehension
Formal Language A language precisely defined by grammar
Machine Independence Programming languages which do not rely on characteristics of a specific machine
High Level Programming Language Programming language that includes a variety of abstractions
Object Oriented Programming (OOP) A programming process in which a software system is viewed as a collection of units (objects), each of which is capable of performing actions immediately related to itself and can request action of other objects
Primitive Data Type A data type built into a programming language
Data Structure The conceptual shape or arrangement of data
Function A set of instructions for performing a task that can be used as an abstract tool by other program units
Local Variable A variable declared within a function
Global Variable A variable whose scope is not restricted to a particular part of a program
Source Program A program in its original form, prior to translation
Object Program The translated version of a program
Data Types:
Integer Numeric data consisting of whole numbers
Float Numeric data that contains values other than whole numbers and probably stored in floating point notation
String A sequence of characters
Python Collection Data Structures
List An ordered, changeable collection used to store multiple items (duplicate varaibles are allowed) in a single variable
Tuple An ordered, unchangeable collection used to store multiple items in a single variable
Set An unordered, unchangeable, unindexed collection used to store multiple items in a single variable
Dictionary An ordered, changeable collection used to store data in key:value pairs
